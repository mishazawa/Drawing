### Source code

`/web` - Next.js app. 

```sh
npm i
npm run build # to build web version
npm run dev   # to start development server
```

`/desktop` - electron wrapper.

```sh
npm  i
npm run make # to build desktop version. by default it lookup the website. can be changed in constants.ts
```
